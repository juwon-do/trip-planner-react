# README

## Trip planner front end

## Installation

Promise based HTTP client for the browser and node.js
``` bash
$ npm install axios
```
Instruction: [Bootstrap](https://getbootstrap.com/docs/5.3/getting-started/introduction/)
``` bash
npm install bootstrap@v5.2.3
```


React google maps api
``` bash
npm i -S @react-google-maps/api
```


## Description
Let people create and sign in to their account and plan their trip.

Users can create trips and places which are belong to a trip.
Each trip can have a budget and a budget can have multiple expenses.

``` bash
import { GoogleMap, LoadScript, InfoWindowF } from '@react-google-maps/api'
```
Used it to show each places on the google map.
Used modal to hide some create actions.
Used Bootstrap to use collapsable button.
